Why Montserrat?

Montserrat delivers excellent geometric legibility across screen sizes. Its open counters and distinct characters make it a perfect match for public service portals, helping to satisfy WCAG 1.4.12 Text Spacing and prevent reading strain.

Base settings: font-family: 'Montserrat'

Heuristic Audit & Critical WCAG Gaps

Audited review of target web properties against cognitive accessibility models.

High Severity WCAG 1.4.3

Contrast Gaps in Original Color Selection

The original brand colors like standard flag yellow fail contrast minimums when displayed as flat textual items on light backgrounds.

System Fix: We strictly mapped the brand palette to high-contrast blue (`#002395` / `#4066E0`) text, reserving yellow (`#F4B41A`) solely for dark backgrounds.
Moderate Severity WCAG 2.4.7

Missing Focus Indicator States

Sequential keyboard navigators cannot see visual highlight indicators on many primary interactives.

System Fix: Enforced prominent high-visibility `:focus-visible` outline styles with customized offsets in custom components.

Typography Rules

Under WCAG 1.4.12, text spacing values are critical. The style guide mandates standard support for customized spacing tweaks:

  • Line height: Enforced minimum of 1.5rem on standard body copy blocks.
  • Paragraph gap: Enforced minimum margin-bottom of 1.5 times the font size.
  • Dyslexia spacing utility adjusts letter and word spacing parameters instantly.
